# Flezi Foundry — Agentic Development Lifecycle (ADLC)

## Overview
Flezi Foundry was launched in May 2026 as an AI-augmented delivery platform. It extends FleziPT into actual software delivery with governed human-agent collaboration models.

## Two Service Modes

### Agentic Development Lifecycle (ADLC)
AI agents across planning, coding, review, testing, security, and documentation — under human supervision:
- **Planning Agent**: Requirements analysis, task decomposition, architecture design
- **Coding Agent** (CodeVista): Code generation, refactoring, bug fixing
- **Review Agent**: Code review, security scanning, best practice enforcement
- **Testing Agent** (TestVista): Test generation, execution, coverage analysis
- **Documentation Agent**: Auto-generation of technical docs, API docs, user guides
- **Orchestration Agent** (AgentVista): Coordinates handoffs between specialized agents
- Target: **30% more output within same budget**

### Agentic Managed Services (AMS)
Operations agents for alert triage, incident resolution, remediation, and service improvement:
- Target: **60-90% automation** of first-line support requests
- Target: **99.5% SLA compliance**

## Four Foundations
1. **Hybrid Sovereign Infrastructure**: Azure VPC + FPT AI Factory (NVIDIA H100/H200 GPUs)
2. **Hybrid FTE Pods**: Human-in-the-Loop / Human-on-the-Loop governance model
3. **Capacity-Based Pricing**: Linked to story-point velocity
4. **Digital Brain & Skill Marketplace**: Institutional knowledge accumulation

## Service-as-a-Software Model
Flezi Foundry operates on a governed delivery model where:
- AI agents handle execution under human supervision
- Human supervisors approve critical decisions
- All actions are transparent and auditable
- Performance is measured against defined KPIs
