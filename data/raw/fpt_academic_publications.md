# FPT Software Academic Research Publications & Open Source Tools (2023–2026)

## 1. Dr. Nghi D. Q. Bui and the FSoft-AI4Code Initiative
Spearheaded by the **FSoft-AI4Code** research division (led by Dr. Nghi D. Q. Bui), FPT Software is a prominent contributor to the global AI for Software Engineering (AI4SE) community. The division focuses on open-source datasets, autonomous coding agents, and Large Language Models (LLMs) specialized in code comprehension and repository-level generation.

## 2. Key Academic Publications

### A. CodeWiki (ACL 2026)
*   **Research Title:** *CodeWiki: Evaluating AI's Ability to Generate Holistic Documentation for Large-Scale Codebases*
*   **Conference:** Published at the Annual Meeting of the Association for Computational Linguistics (ACL 2026).
*   **Authors:** Hoang, A. N., Le-Anh, M., Le, B., & Bui, N. D. Q.
*   **Key Contribution:** An open-source framework and benchmark designed to evaluate how generative AI models construct repository-level, structured, and cross-referenced documentation for multi-language software projects. The associated repository has gathered over 1,170 GitHub stars.
*   **Link:** https://github.com/FSoft-AI4Code

### B. AgileCoder (FORGE 2025)
*   **Research Title:** *AgileCoder: Dynamic Collaborative Agents for Software Development based on Agile Methodology*
*   **Conference:** Featured at the 2025 International Conference on Software Engineering (ICSE) Workshop on Foundation Models for Software Engineering (FORGE 2025).
*   **Authors:** Nguyen, M. H., Chau, T. P., Nguyen, P. X., & Bui, N. D. Q.
*   **Key Contribution:** Formulates a multi-agent framework where agents adopt roles from the Agile Scrum methodology (Product Owner, Scrum Master, Developer, Tester) to collaboratively design, code, and test complex software applications.
*   **Verified Metrics:** Achieved a **79.27% pass@1 on HumanEval** and **84.31% pass@1 on MBPP**, outperforming traditional monolithic coding pipelines and multi-agent frameworks like MetaGPT.
*   **Link:** https://github.com/FSoft-AI4Code/AgileCoder

### C. HyperAgent (arXiv 2024)
*   **Research Title:** *HyperAgent: Generalist Software Engineering Agents for Repository-Level Bug Fixing*
*   **Publication:** arXiv preprint arXiv:2409.16299 (2024).
*   **Key Contribution:** A generalist coding agent optimized for autonomous repository-level bug fixing, program repair, and feature addition.
*   **Verified Metrics:** Resolved **31.4% of SWE-bench-Verified** issues and successfully repaired **249 bugs on Defects4J**.
*   **Link:** https://github.com/FSoft-AI4Code/HyperAgent

### D. TheVault (EMNLP 2023)
*   **Research Title:** *TheVault: A Multilingual Code Dataset for Code Search and Comprehension*
*   **Conference:** Published at the Conference on Empirical Methods in Natural Language Processing (EMNLP 2023).
*   **Key Contribution:** A massive, clean, and deduplicated multilingual dataset of source code and natural language docstrings to train LLMs in code search, code summary, and translation. The project has over 105 GitHub stars.
*   **Link:** https://github.com/FSoft-AI4Code/TheVault

### E. RepoHyper (FORGE 2025)
*   **Research Title:** *RepoHyper: Context-Aware Graph-Based Repository-Level Code Completion*
*   **Conference:** Featured at FORGE 2025.
*   **Key Contribution:** Implements graph neural networks and semantic graphs to perform precise context retrieval for repository-level code completion. (73 GitHub stars).
*   **Link:** https://github.com/FSoft-AI4Code/RepoHyper
