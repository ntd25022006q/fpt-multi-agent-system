# FPT Software AI Research Lab & Academic Contributions (2023–2026)

## 1. FPT Software AI Research Lab (FSoft AI4Code)
FPT Software established its dedicated AI Research Laboratory under the brand **FSoft AI4Code**, publishing top-tier research at premier venues including ACL, EMNLP, ICSE, NeurIPS, and ICLR.

### 1.1 AgileCoder — Multi-Agent Coding System
*   **What it does:** A multi-agent coding system where specialized agents collaboratively generate, review, test, and refactor code.
*   **Performance:** Achieved **79.27% HumanEval pass@1** and **84.31% MBPP pass@1**, outperforming MetaGPT on both benchmarks.
*   **Publication:** Accepted at FORGE 2025 (ICSE workshop on LLMs for Software Engineering).
*   **GitHub:** https://github.com/FSoft-AI4Code/AgileCoder (950+ stars)

### 1.2 HyperAgent — Repository-Level Software Engineering Agent
*   **What it does:** An autonomous end-to-end software engineering agent designed to solve real-world GitHub issues across large codebases.
*   **Performance:** Resolved **31.4%** of SWE-Bench-Verified issues; fixed **249 bugs** from the Defects4J benchmark.
*   **Publication:** arXiv:2409.16299
*   **GitHub:** https://github.com/FSoft-AI4Code/HyperAgent

### 1.3 CodeWiki — Repository-Level Documentation Generation
*   **What it does:** Automatically generates holistic, structured documentation for entire GitHub repositories.
*   **Awards:** 1,170+ GitHub stars.
*   **Publication:** Accepted at ACL 2026 (Annual Meeting of the Association for Computational Linguistics).
*   **GitHub:** https://github.com/FSoft-AI4Code

### 1.4 RepoHyper — Graph-Based Code Completion
*   **What it does:** End-to-end repository-level code completion using graph neural networks.
*   **Impact:** 73 GitHub stars; accepted at FORGE 2025.
*   **GitHub:** https://github.com/FSoft-AI4Code/RepoHyper

### 1.5 TheVault — Multilingual Code Dataset
*   **What it does:** A large-scale multilingual code dataset for training and fine-tuning code LLMs across 10+ programming languages.
*   **Awards:** 105 GitHub stars.
*   **Publication:** EMNLP 2023.
*   **GitHub:** https://github.com/FSoft-AI4Code/TheVault
